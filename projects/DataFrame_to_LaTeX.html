DataFrame转LaTeX

物理实验的数据一般记在excel表格里，但是报告用LaTeX写，所以写了这段代码把excel转为LaTeX。

一个注意事项是数据要贴在excel表格左上方，且不要编辑数据以外的任意单元格，否则行列数计算会出错。

# 20240317
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 唯一需要改的地方，把路径替换为你自己的文件路径
df=pd.read_excel("some_data.xlsx", header=None)
print('\\begin{table}')
print('\\centering')
print('\\caption{*to be edited*}')
string='{|'
for i in range(df.shape[1]):
    string+='c|'
string+='}'
print('\\begin{tabular}'+string)

print('\\hline')
for index, row in df.iterrows():
    string=''
    for cell in row:
        if pd.isna(cell):
            string+='{} & '.format('')
        else:
            string+='{} & '.format(cell)
    string=string[:-2]
    string+='\\\\'
    print(string)
    print('\\hline')
print('\\end{tabular}')
print('\\end{table}')
