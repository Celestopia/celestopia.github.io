Quick Power Algorithm

Question

Given very large positive intergers $a$, $b$, $c$, how to calculate

$$a^b \mod c$$

Mathematical Basis

$$(a * b ) \mod c = \Big((a \mod c) * (b \mod c)\Big) \mod c$$

Python Implementation

def quick_pow(a, b, c): # a^b mod c
    r=1
    for i in range(0,b):
        r=((a%c)*(r%c))%c
    return r

Example:

a=2790
b=413
c=3233
print(quick_pow(a,b,c)) # Output: 65
